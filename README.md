# My React Portfolio

Welcome to My React Portfolio! This portfolio showcases my projects and skills as a front-end developer.

## Getting Started

To run this portfolio locally, follow these steps:

1. Clone this repository.
2. Navigate to the project directory.
3. Install dependencies: `npm install`.
4. Start the development server: `npm start`.

Visit [https://aijus2022.github.io/my-frontend-dev-portfolio-in-react/](https://aijus2022.github.io/my-frontend-dev-portfolio-in-react/) to view the live version of this portfolio.
